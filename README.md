# 🌍 Easy Go - Travel Management Website

## 📌 Project Description

Easy Go is a travel management website designed to make trip planning simple and user-friendly. It allows users to search flights, explore hotels, view travel packages, and plan trips based on their preferences and budget.

---

## 🚀 Features

* ✈️ Flight search and booking
* 🏨 Hotel listings based on location and budget
* 🎁 Travel packages and seasonal deals
* ❤️ Wishlist feature to save favorite places
* 🧭 Personalized travel plans based on user mood
* 🌍 Explore multiple countries and destinations
* 🏆 Travel rewards system
* 💬 AI chatbox for quick help
* 🎮 Travel games and quiz
* 🌙 Dark mode UI

## ⚙️ How to Run the Project

1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder into `htdocs`
4. Open browser and go to:
   [http://localhost/Easy-Go](http://localhost/Easy-Go)
